Licensing

Clear, straightforward licensing options designed to fit your needs. From individual designers to large studios, we keep it simple and fair.

Desktop License

Perfect for print design, branding, and desktop applications.

  • Install on up to 5 devices
  • Use in print and digital publications
  • Create logos and branding materials
  • Include in client presentations
  • No web embedding or app distribution

Web License

For websites and web applications using @font-face or web font services.

  • Self-hosted or CDN delivery
  • Unlimited page views
  • Multiple domains allowed
  • Includes all weights and styles
  • No desktop installation rights

App License

For mobile apps, desktop applications, and software products.

  • Embed in iOS, Android, and desktop apps
  • Unlimited app downloads
  • Multiple platforms supported
  • No web or print usage rights
  • Source code protection included
